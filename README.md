# Car-rental-system-1
该汽车租赁系统基于springboot+vue技术开发。后端springboot处理车辆信息存储、订单管理、用户信息管理等业务逻辑，确保系统稳定可靠。前端vue构建出简洁美观的界面，具有良好的用户体验。用户可以在系统中查看车辆信息，筛选心仪车辆，进行预订、还车操作，以及管理自己的租赁订单，为用户提供便捷、高效的汽车租赁服务。
